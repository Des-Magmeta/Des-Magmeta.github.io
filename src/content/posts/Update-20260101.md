2026 年 1 月 1 日零时，Des-Magmeta 在 GitHub 平台上发布了「游趣体」0.800 版本更新。此次更新主要补全了《通用规范汉字表》，并优化了相当多的字符，以及少许细节调整。除此之外，OpenType 特性也发生了相应变动。值得注意的是，为精简字体文件，此版本移除了较不常用的「楔形标点」`ss06`特性。需使用此特性的用户，请回退到上一版本。

各位用户可通过 GitHub 项目获取最新版本的字体：
::github{repo="Des-Magmeta/ChildFunSans"}

需要注意的是，此前作者制作简体子集版时，根据 FontCreator 语言名称缩写确定，其中简体中文代码为 ZHS。今根据语言缩写，分别取「**Ch**inese, **S**implified」的前两个字母和首字母，修改子集版名称后缀为 -CHS。因此两种规模的文件中文名称均不变，但英文名称发生改动，在操作系统中可能出现将新旧版本视为两款不同字体的问题。故作者建议，如需使用最新版本，请先卸载旧版文件后再安装！  


## 完整版更新详情
- 对字体名称作以下变更，便于后期开发：

原始名称 | Original name | 新名称 | New name
---- | ---- | ---- | ----
游趣体 简 | Child Fun Sans ZHS | 游趣体 CHS | Child Fun Sans CHS 

- 版权信息部分年号改为 2026； 
- 修改「′`U+2032`」「″`U+2033`」「∓`U+2213`」的宽度，并将原有的全宽符号设为通过「全角字符」`fwid`特性调用，相关联的特性进行相应调整；
- 调整「标点挤压」`chws`、「半角替代」`halt`、「基本附属标记」`mark`等若干 OpenType 特性；
- 修复部分字符轮廓问题、「堍」「褟」字形错误问题、「繇」包含部件漏改问题；
- 重制包含「车」「东」等部件汉字，并优化上一版本部分字形约 520 字；
- 补齐《通用规范汉字表》所收字符，大约 925 字；
- 参照「文津宋体」，新增部分注音符号和附标符号结合的组合形，并针对小型注音符号新增竖排特性。

## 简体版更新详情
- 将「≤`U+2264`」「≥`U+2265`」显示与「⩽`U+2A7D`」「⩾`U+2A7E`」同形；
- 调整「㊆`U+3286`」「苡`U+82E1`」字形。
- 进一步筛选字符，移除了部分不常用的西文字母与符号<sub>（如带 tonos 的希腊字母，假名连字字符，方言注音符号等）</sub>。

受限于文章篇幅，更新详情不在此尽数列出。如需了解，请前往源项目中的`Docs\Update.md`阅读完整内容。
